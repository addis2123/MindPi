## Raspberry Pi boot partition files

Copy these files to the FAT32 boot partition of the SD card:

- `bootcode.bin`
- `start.elf`
- `fixup.dat`
- `config.txt` (from this repository's `boot/config.txt`)
- `kernel8.img` (generated by `make`)

The `bootcode.bin`, `start.elf`, and `fixup.dat` files come from Raspberry Pi firmware releases.
